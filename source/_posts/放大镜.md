# 放大镜

## 结构

```js
1. 大体结构就是 一个大盒子里面有两个小盒子
2. 这两个小盒子 是水平排列
3. 左侧盒子 是缩略图 此图片比较小 上面会有一个遮罩层
4. 右侧盒子是一个详情图 要比左侧的图片要大 这里要设置超出隐藏
5. 当我们鼠标进入左侧时候 才会显示我们的mark层 和 右侧的详情图
```

## 思路

```js
1. 首先我们鼠标进入到缩略图的时候 我们的mark 要现实出来 并且位置要在鼠标的中间
2. 当我们mark层 随着鼠标移动的时候 右侧的详情图 也要随着鼠标移动
3. 当我们的mark层 移动到缩略图边缘的时候 不能超出去
```

## 鼠标进入和离开

```js
1.我们鼠标进入到我们的缩率图的时候 遮罩层 和 详情图 都要显示出来
2.我们鼠标离开我们的缩略图的时候 遮罩层 和 详情图 都要隐藏
```

```js
$(function(){
    //1.获取我们要操作的元素
    //最外层的大盒子
    let $magnifier = $('.magnifier')
        //大盒子下的所旅途的盒子
        $abbre = $magnifier.find('.abbre')
        // 缩略图中的遮罩层
        $mark = $magnifier.find('.mark')
        // 缩略图中的详情
        $detail = $magnifier.find('.detail')
        // 详情中的图片
        $detailIMG = $detail.find('img')
    //2.鼠标进入abbre的时候 mark显示 详情要显示 鼠标移除的时候 要隐藏
    $abbre.mouseenter((function(ev){
        //鼠标进入到abbre
        //3. 让mark 和 detail 显示
        $mark.css('display','block')
        $detail.css('display','block')
    }))
    .mousemove(function(ev){
        //鼠标在abbre移动
    })
    .mouseleave(function(ev){
        //鼠标离开 让mark 和 detail 隐藏
        $mark.css('display','none')
        $detail.css('display','none')
    })
})
```

## 动态记录大图的大小

```js
1. 我们要用关系比例 计算出 我们大图的大小
2. 关系 就是 用我们的 mark / 缩略图 = 大图 / 大图边界
3. 大图 = mark * 大图边界 / 缩略图
4. 这里我们的目的就是 为了让我大图里面那一块的显示 和 我们的mark 是成比例的
```

```js
$(function(){
    //1.获取我们要操作的元素
    //最外层的大盒子
    let $magnifier = $('.magnifier')
        //大盒子下的所旅途的盒子
        $abbre = $magnifier.find('.abbre')
        // 缩略图中的遮罩层
        $mark = $magnifier.find('.mark')
        // 缩略图中的详情
        $detail = $magnifier.find('.detail')
        // 详情中的图片
        $detailIMG = $detail.find('img')

    //1.动态计算大图的大小 为了让我们mark 和 我们的大图 显示成比例
    let abbreW = $abbre.width()
        abbreH = $abbre.height()
        markW = $mark.width()
        markH = $mark.height()
        detailW = $detail.width()
        detailH = $detail.height()
        detailIMGW = 0
        detailIMGH = 0
        detailIMGW = detailW / (markW / abbreW)
        detailIMGH = detailH / (markH / abbreH)
        $detailIMG.css({
            width:detailIMGW,
            height:detailIMGH
        })
    $abbre.mouseenter((function(ev){
        $mark.css('display','block')
        $detail.css('display','block')
    }))
    .mousemove(function(ev){
    })
    .mouseleave(function(ev){
        $mark.css('display','none')
        $detail.css('display','none')
    })
})
```

## 鼠标控制mark 移动

```js
1. mark 要是移动的话 我们首先要知道 鼠标的位置 
2. mark 要在鼠标中心的话 我们还要知道 mark的宽和高
3. mark 的 位置 应该 鼠标到页面的距离 -  abbre 到页面的距离 - mark/2
```

![](2.png)

```js
$(function(){
    let $magnifier = $('.magnifier')
        $abbre = $magnifier.find('.abbre')
        $mark = $magnifier.find('.mark')
        $detail = $magnifier.find('.detail')
        $detailIMG = $detail.find('img')

    let abbreW = $abbre.width()
        abbreH = $abbre.height()
        //1.获取abbre的距离上边 和 左边的距离 是个对象
        abbreOffset = $abbre.offset()
        markW = $mark.width()
        markH = $mark.height()
        detailW = $detail.width()
        detailH = $detail.height()
        detailIMGW = 0
        detailIMGH = 0
        detailIMGW = detailW / (markW / abbreW)
        detailIMGH = detailH / (markH / abbreH)
        $detailIMG.css({
            width:detailIMGW,
            height:detailIMGH
        })
        //1.动态计算mark移动的距离 和 大图移动的位置
        const computed = (ev) => {
            //鼠标距离页面的距离 减去 abbre 的左偏移 在 减去 mark的宽的一半
            let curL = ev.pageX - abbreOffset.left - markW /2
                curT = ev.pageY - abbreOffset.top - markH /2
            //需要做边界判断 如果 你mark的 在左边超出了abbre 我们就归零 如果在右边 超出了安全距离 就让他等于我们的安全距离
            let minL = 0
                minT = 0
                maxL = abbreW - markW  
                maxT = abbreH - markH 
                curL = curL < minL? minL : (curL > maxL?maxL:curL)
                curT = curT < minT? minT : (curT > maxT?maxT:curT)
            $mark.css({
                left:curL,
                top:curT
            })
        }
    $abbre.mouseenter((function(ev){
        $mark.css('display','block')
        $detail.css('display','block')
        computed(ev)
    }))
    .mousemove(function(ev){
        computed(ev)
        
    })
    .mouseleave(function(ev){
        $mark.css('display','none')
        $detail.css('display','none')
    })
})
```

## 大图移动

```js
1. 我们的mark 移动多少 大图就移动多少
2. 但是 我们mark 向右移动 我们的大图是像左移动的 正好相反
3. 另外 我们大图移动的距离 也是要根据比例去缩放的
a(mark 在 abbre移动中的距离) / abbre = b(大图要移动的距离) /大图的尺寸
b = a * 大图尺寸 / abbre
```

```js
//5.动态计算mark移动的距离 和 大图移动的位置
        const computed = (ev) => {
            //鼠标距离页面的距离 减去 abbre 的左偏移 在 减去 mark的宽的一半
            let curL = ev.pageX - abbreOffset.left - markW /2
                curT = ev.pageY - abbreOffset.top - markH /2
            //需要做边界判断 如果 你mark的 在左边超出了abbre 我们就归零 如果在右边 超出了安全距离 就让他等于我们的安全距离
            let minL = 0
                minT = 0
                maxL = abbreW - markW  
                maxT = abbreH - markH 
                curL = curL < minL? minL : (curL > maxL?maxL:curL)
                curT = curT < minT? minT : (curT > maxT?maxT:curT)
            $mark.css({
                left:curL,
                top:curT
            })
            //根据公式去算 大图应该移动的距离的比例 
            // a(mark移动的距离) / abbre = b(大图移动的距离) / 大图的尺寸
            // 如果mark 向左移动 那么 大图应该向右移动
            $detailIMG.css({
                left:-curL / abbreW * detailIMGW,
                top:-curT / abbreH * detailIMGH
            })
        }
```
